Properties in Project Files‚Äã

Bool‚Äã

Implicit:

"Anchored": false

Explicit:

"Anchored": {
    "Bool": false
}

String‚Äã

Implicit:

"Text": "Hello, world!"

Explicit:

"Text": {
    "String": "Hello, world!"
}

Number Types (Int32, Int64, Float32, Float64)‚Äã

Implicit:

"RespawnTime": 12.5
"ZIndex": 70

Explicit:

"RespawnTime": {
    "Float32": 12.5
}
"ZIndex": {
    "Int32": 70
}

Enum‚Äã

Implicit enum values can only be used for enums created before the current release of Rojo. To specify an enum that was recently released, used the explicit property syntax.

Implicit:

"Shape": "Cylinder"

Explicit:

"Shape": {
    "Enum": 2
}
